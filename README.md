Personal Profile Page

Description

This is a simple personal profile webpage created as a mini project for a bootcamp.  
The goal of this project is to practice basic HTML structure and commonly used tags.

The page presents basic personal information and includes a simple contact form.

Features
- Displays full name using a heading
- Short bio section describing the author
- Profile image
- Contact form with:
  - Name input
  - Email input
  - Message textarea
  - Submit button

Technologies Used
- HTML5
- CSS
